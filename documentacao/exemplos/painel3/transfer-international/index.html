Transferência

Para iniciar o processo de transferência do domínio adidas.com para outro provedor, siga as instruções abaixo:

Desabilite a trava de transferência

A trava garante que o domínio internacional não seja transferido para outro provedor. Caso queira iniciar um processo de transferência, desabilite a trava.

Gere uma senha de transferência (Authorization Code / AuthCode)

Gerar senha

Uma cópia da senha gerada será enviada para o e-mail email@dominio.com.br e deverá ser informada ao novo provedor.

Importante: Caso existam outros domínios vinculados ao [ID_USUARIO] eles também serão transferidos. Para mais informações acesse a FAQ do Registro.br
